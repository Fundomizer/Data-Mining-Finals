# Data-Mining-Finals

Git Repository in compliance to our Final Project requirements

## Description

This data mining activity follows the CRISP-DM framework to perform data mining on the following dataset taken from Kaggle:

https://www.kaggle.com/datasets/ualex1/synthetic-erp-sales-dataset/data

Clustering and Associative-Rule Mining techniques were applied to the dataset to achieve the following business objectives:
- Maximize Customer Lifetime Value 
- Optimize Product Mix and Cross-Selling 
- Increase Revenue per Customer Segment
- Mitigate Customer Attrition
- Strengthen Sales Forecasting and Seasonal Planning
- Improve Store and Category Performance Visibility
